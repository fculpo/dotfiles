# AeroSpace Keybindings Reference

## Main Mode

| Binding | Action |
|---|---|
| `alt + enter` | Open WezTerm |
| `alt + h/j/k/l` | Focus left/down/up/right |
| `alt + shift + h/j/k/l` | Move window left/down/up/right |
| `alt + 1-9` | Switch to workspace 1-9 |
| `alt + shift + 1-9` | Move window to workspace 1-9 |
| `alt + f` | Toggle fullscreen |
| `alt + slash` | Cycle tile layout (horizontal/vertical) |
| `alt + comma` | Cycle accordion layout |
| `alt + minus/equal` | Resize window smaller/larger |
| `alt + tab` | Switch to previous workspace |
| `alt + shift + tab` | Move workspace to next monitor |
| `alt + shift + ;` | Enter service mode |

## Service Mode

Enter with `alt + shift + ;`, all commands return to main mode.

| Binding | Action |
|---|---|
| `esc` | Reload config |
| `r` | Reset layout (flatten workspace tree) |
| `f` | Toggle floating/tiling |
| `b` | Balance window sizes |
| `backspace` | Close all windows except current |
| `alt + shift + h/j/k/l` | Join with left/down/up/right |

## Workspace Assignments

| Workspace | Apps |
|---|---|
| 1 | Safari, Chrome, Arc, Firefox |
| 2 | WezTerm |
| 3 | VS Code |
| 4 | Slack, Discord, Messages |
| 5 | Finder |
| 6-9 | Unassigned |

System Preferences always floats.
